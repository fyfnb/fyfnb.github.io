前后端分离

前后端分离就是将一个应用的前端代码和后端代码分开写。

传统的javaweb开发中，前端使用jsp进行开发，jsp不是由后端开发着来独立完成的。

前端->HTML->后端->jsp，传统后端的工作是把前端给过来的东西嵌套到jsp上，开发效率极低。于是就出现了前后端分离的方式进行开发，前端独立编写客户端代码，后端也只需要独立编写服务端代码提供数据接口就可以了，前端通过ajax请求来访问后端的接口，将Model展示到View中即可。

前后端开发者只需要提前约定好接口文档（URL，参数，数据类型等等），然后分别独立开发即可，前端可以造假数据进行测试，完全不需要依赖于后端，后端只需要能返回所需要的数据就可以，最后完成前后端集成即可，真正实现了前后端应用的解耦合，极大的提升了开发的效率。

前端应用：负责数据展示和用户交互

后端应用：负责数据处理接口

前端HTML通过Ajax->RestFul后端数据接口。

前后端分离就是将一个单体应用拆分成两个独立的应用，前端应用和后端应用以JSON格式进行数据交互。

Ajax&JQuery

什么是Ajax

  • Asynchronous JavaScript and XML 异步的js和xml
  • ajax是一种在无需重新加载整个网页的情况下，能够更新部分网页的技术
  • Ajax不是一种新的编程语言，而是一种用于创建更好更快的以及交互性更强的web应用程序的技术

Ajax的使用： 百度的搜索栏输入一个字段后不需要提交就会自动给你找到相关的字段补全你的搜索内容。

请求类型为Type = xhr为ajax请求

特点

  • 传统的网页想要提交一个表单，都要重新加载整个网页
  • 采用ajax技术的网页，通过在服务器后端进行少量的数据交互，就可以实现局部的异步更新
  • ajax技术可以创建更丰富、更好用的web页面。

JQuery.ajax

  • Ajax的核心是XMLhttpRequest对象XHR。XHR为向服务器发送请求和解析服务器响应提供了接口，能够以异步方式从服务器获取新数据。
  • jQuery提供多个和Ajax有关的方法
  • 通过jQuery AJAX方法 可以使用Http Get和Post从远程服务器上请求文本HTML、XML或者JSON，同时可以把这些外部数据直接载入网页的备选元素中
  • jQuery不是生产者，只是搬运工
  • jQuery Ajax本质就是XMLHttpRequest，对他进行了封装，方便调用。
  • jQuery是一个库，存放着js的大量函数。

Vue集成Element UI

Element UI后台管理系统的主要标签：

  • el-container：构建起整个页面的框架
  • el-aside：构建起左侧菜单栏
    • el-menu：菜单栏中的内容的常用属性
      • default-openeds：默认展开的菜单，通过菜单的index值来关联，传递要打开的index值。
      • default-active：默认选中的菜单。
    • el-submenu：可展开的菜单。
      • index下标，是文本类型，传递的时候要用数组和字符串进行传递。
    • template：对应el-submenu的菜单名
    • i：设置菜单的图标，通过class属性设置，不同的图标名对应不同的图标
      • el-icon-message
      • el-icon-menu
      • el-icon-setting
    • el-menu-item：菜单的子节点，不可以再展开，常用属性：index
